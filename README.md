# Students-Performance_Analysis
Explored Data Analysis on Student Performance Dataset
This project analyzes student performance using math, reading, and writing scores.

## Objective
To explore relationships between different academic subjects using data visualization.

## Dataset
Student Performance Dataset (Kaggle)

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Visualizations
- Correlation heatmap of scores
- Scatter plot between math and reading scores

## Key Insights
- Strong positive correlation between reading and writing scores
- Math scores also show a positive relationship with reading performance
